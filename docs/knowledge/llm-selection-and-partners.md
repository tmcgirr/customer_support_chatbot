# LLM Selection and Partners

Cadre doesn't use one model for every situation. Model selection depends on the use case, answer quality, latency, cost, context requirements, integration needs, deployment constraints, and data-handling requirements.

Because these factors differ from one workflow to the next, the right model for one problem is often not the right model for another. A final recommendation normally requires understanding your workflow and your security requirements.

Cadre works across providers and platforms, including:

- OpenAI
- Anthropic (Claude)
- Google
- Microsoft
- AWS
- Salesforce
- Snowflake

Cadre also uses OpenRouter for flexible model access.

To get a recommendation tailored to your workflow and security needs, you can request a conversation with an AI strategist.
