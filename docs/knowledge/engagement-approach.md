# How Cadre Runs Engagements

Cadre often starts with the **AI Transformation Intensive**, a structured program that takes a team from zero clarity to a prioritized roadmap they can execute. It runs department by department to identify high-ROI use cases, then moves into implementation, training, and change management.

The work is organized around Cadre's **eight-pillar framework** for AI transformation:

1. Build a dedicated AI team
2. Deploy an AI Command Center
3. Create an AI-first culture shift
4. Connect and enable the tech stack
5. Assess AI-healthy data
6. Build an AI agent readiness framework
7. Run departmental AI deep dives
8. Set a three-year AI vision

The **AI Command Center** gives an organization a centralized, secure place to run and govern its AI tools, agents, training, and results, instead of fragmented, ad-hoc AI use across teams.

Beyond the intensive, the specific scope of an engagement still depends on the business problem — the number of workflows, the systems involved, data requirements, implementation complexity, and the level of organizational support required — so a strategist would need to understand your systems, data, and constraints before defining the exact engagement.

The best way to explore what an engagement could look like for you is to request a conversation with an AI strategist.
